# proyecto-investigacion-ember
Este es un proyecto de investigación universitario sobre el framework.

Se buscó desarrollar la simulación de una página para una clínica dental, en la cual se encuentra información ficticia sobre esta y los servicios que ofrecen.
A su vez, se implementó la funcionalidad de agendar citas según el servicio que se requiera, sin embargo, los datos funcionan de manera volátil al ser una demo la funcionalidad.
